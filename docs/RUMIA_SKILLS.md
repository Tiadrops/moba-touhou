## Qスキル「通常弾幕1」

プレイヤー方向に向けて予告線を順次表示し、弾幕を発射する基本攻撃スキル。
**CC（スタン）で中断可能。**

### パラメータ

| 項目 | 値 |
|------|-----|
| 詠唱時間 | 1400ms (7本 × 200ms) |
| クールダウン | 5000ms |
| 基礎ダメージ | 30 |
| ATK係数 | 0.5 (ATK × 50%) |

### 弾幕パラメータ

| 項目 | 値 |
|------|-----|
| 予告線の本数 | 7本 |
| 予告線の出現間隔 | 200ms |
| 予告線から弾発射までの遅延 | 300ms |
| 縦方向の弾数 | 10発/列 |
| 縦方向の弾間隔 | 40px |
| 横方向の弾間隔 | 30px |
| 弾速 | 1100 px/s (20m/s) |
| 弾の半径 | 13.75px (0.25m) |
| 弾タイプ | KSHOT.MEDIUM_BALL.RED |
| 表示スケール | 0.06 |

### 予告線ごとの弾数

| 予告線番号 | 横方向の列数 |
|-----------|-------------|
| 1本目 | 1列 |
| 2-3本目 | 2列 |
| 4-6本目 | 3列 |
| 7本目 | 4列 |

### 挙動

1. **詠唱フェーズ**: 200ms間隔で予告線が順次出現（各予告線はプレイヤー方向を狙う）
2. **発射フェーズ**: 各予告線の出現から300ms後に弾幕発射
3. 弾は「八の字」に広がるように角度がつく（各列が中心から3度ずつ広がる）
4. CCで中断された場合、即座にクールダウンに入る

## Wスキル「通常弾幕2」

### パラメータ

| クールダウン | 6000ms |
| 基礎ダメージ | 30 |

### 弾幕パラメータ

| 項目 | 値 |
|------|-----|
| リング数 | 3 |
| 1リングあたりの弾数 | 20発 |
| 弾の半径 | 13.75px (0.25m) |

| 表示スケール | 0.06 |

| リング2の弾速 | 550 px/s (10m/s) |

### 挙動

1. **詠唱フェーズ**: 500msの詠唱時間
2. **発射フェーズ**: 3つのリング弾幕を同時に発射

## Eスキル「通常弾幕3」

ルーミアが1秒かけてランダム方向に移動しながら、2列の弾を配置して発射する。
**CC（スタン）で中断可能（ブレイク）。**

### パラメータ

| 項目 | 値 |
|------|-----|
| 詠唱時間 | 500ms |
| クールダウン | 6000ms |
| 基礎ダメージ | 20 |
| ATK係数 | 0.3 (ATK × 30%) |

### 移動パラメータ

| 項目 | 値 |
|------|-----|
| 移動時間 | 1000ms (1秒) |
| 移動距離 | 275px (5m) |
| 移動方向 | ランダム（プレイエリア内に収まるよう調整） |

### 弾幕パラメータ

| 項目 | 値 |
|------|-----|
| 弾列数 | 2列（移動方向の前方に配置） |
| 1列あたりの弾数 | 20発 |
| 弾配置間隔 | 50ms |
| 列の間隔 | 55px (1m) |
| 弾の半径 | 13.75px (0.25m) |
| 弾タイプ | KSHOT.RINDAN.GREEN（輪弾・緑） |
| 表示スケール | 0.1 |

### 弾速パラメータ（山型）

| 弾のインデックス | 弾速 |
|----------------|------|
| 1発目 | 165 px/s (3m/s) |
| 2発目 | 192.5 px/s (3.5m/s) |
| ... | +27.5 px/s ずつ増加 |
| 11発目（ピーク） | 440 px/s (8m/s) |
| 12発目 | 412.5 px/s (7.5m/s) |
| ... | -27.5 px/s ずつ減少 |
| 20発目 | 192.5 px/s (3.5m/s) |

### 挙動

1. **詠唱フェーズ**: 500msの詠唱時間
2. **移動フェーズ**: 1秒かけてランダム方向に5m移動
   - 移動中も詠唱中判定（CCで中断可能）
3. **弾配置**: 移動中、50ms間隔で2列の弾を配置
   - 弾は移動方向の前方に2列配置（列の間隔1m）
   - 各弾はプレイヤー方向に発射
4. **弾速**: 山型（1発目→11発目で最速→18発目で減速）
5. CCで中断された場合、即座にクールダウンに入る（ブレイク）

### 備考

- 移動弾幕パターン
- 弾速が山型になることで、中央付近の弾が追い越し、弾の密度が変化する
- 移動方向はプレイエリア内に収まるよう自動調整される

## スキル使用AI

### 優先順位

1. **Wスキル**: 最優先で使用（Qスキルが使用中でない場合）
2. **Qスキル**: Wスキルが使用中でない場合に使用
3. **Eスキル**: 他のスキルが使用中でない場合に使用

### 初期クールダウン

| スキル | スポーン後の待機時間 |
|--------|---------------------|
| Qスキル | 1000ms |
| Wスキル | 2000ms |
| Eスキル | 3000ms |

### 移動AI

- プレイエリア上部1/4の高さを目標位置として移動
- プレイヤーのX座標に緩やかに追従（速度50%）
- スキル使用中は移動停止
- 移動範囲はプレイエリア上半分に制限

#### 弾パラメータ

| 項目 | 値 |

#### 弾速パラメータ

**行き（outgoing）**
| 項目 | 値 |
|------|-----|
| 弾速 | 825px/s (15m/s) |

**帰り（returning）**
| 項目 | 値 |
|------|-----|
| 初速 | 33px/s (0.6m/s) |

### Wスキル「アレンジドフォックスファイア」

自身を中心に円形フィールドを展開し、プレイヤーが赤エリア内に入った場合、追尾弾を発射する。

#### パラメータ

| 項目 | 値 |
|------|-----|
| 初期詠唱時間 | 100ms (0.1秒) |
| クールダウン | 8000ms (8秒) |
| 基礎ダメージ | 35 |
| ATK係数 | 0.5 (ATK × 50%) |
| CC中断 | 可能（黄色・赤フェーズ両方） |

#### フィールドパラメータ

| 項目 | 値 |
|------|-----|
| フィールド半径 | 247.5px (4.5m) |
| フィールド総持続時間 | 2000ms (2秒) |
| 黄色フェーズ | 600ms (0.6秒) - キャスト扱い |
| 赤フェーズ | 1400ms (1.4秒) - プレイヤー判定 |

#### 追尾弾パラメータ

| 項目 | 値 |
|------|-----|
| 弾数 | 3発 |
| 弾の半径 | 27.5px (0.5m) |
| 弾タイプ | KSHOT.LARGE_BALL.ORANGE（オレンジ大玉） |
| 表示スケール | 0.11 |
| 弾速 | 660px/s (12m/s) |
| 旋回速度 | 360度/秒 |
| 追尾時間 | 3000ms (3秒) |

#### MSバフパラメータ

| 項目 | 値 |
|------|-----|
| バフ量 | +40% |
| 持続時間 | 2000ms (2秒) |

#### 挙動

1. **詠唱**: 0.1秒の初期詠唱
2. **発動**: 詠唱完了と同時にMSバフ+40%を獲得（2秒間持続）
3. **黄色フェーズ** (0.6秒): 自身を中心に半径4.5mの**黄色**フィールドを展開
   - この間はキャスト扱い
   - CCを受けると**Break**となりスキル終了
4. **赤フェーズ** (1.4秒): フィールドが**赤色**に変化
   - プレイヤーが赤エリア内に入った瞬間、追尾弾3発を発射
   - 発射と同時にスキル終了
   - プレイヤーが入らなければ、2秒後にスキル終了（追尾弾なし）
5. **追尾弾**: プレイヤー方向に3発の追尾弾を発射
   - 各弾は-30度、0度、+30度の角度で発射
   - 3秒間プレイヤーを追尾した後、直進に切り替え

#### 備考

- 黄色フェーズ中はキャスト扱いのためCC（スタン）でBreak可能
- 赤フェーズ中もキャスト扱いのためCC（スタン）でBreak可能
- MSバフはスキル終了後も残り時間がある限り継続
- 追尾弾は画面外に出るか、プレイヤーに命中すると消滅
- MSバフはスロウなどのデバフと共存可能（バフ→デバフの順で計算）

### Eスキル「カーチスクロス」

ルーミアが十字架型の弾幕を纏い、プレイヤー方向に高速突進する。突進終了後、弾幕は同じ方向・速度で画面外まで飛んでいく。

#### パラメータ

| 項目 | 値 |
|------|-----|
| キャスト時間 | 400ms (0.4秒) |
| クールダウン | 6000ms (6秒) |
| 基礎ダメージ | 40 |
| ATK係数 | 0.5 (ATK × 50%) |
| CC中断 | 可能（詠唱中のみ） |

#### 突進パラメータ

| 項目 | 値 |
|------|-----|
| 突進距離 | 385px (7m) |
| 突進速度 | 770px/s (14m/s) |
| 突進時間 | 約0.5秒 (7m ÷ 14m/s) |
| 壁判定 | あり（プレイエリア端で停止） |

#### 十字架弾幕パラメータ

| 項目 | 値 |
|------|-----|
| 弾数 | 6発（十字架配置） |
| 弾の半径 | 16.5px (0.3m) |
| 弾の間隔 | 55px (1m) |
| 弾タイプ | KSHOT.MEDIUM_BALL ID 8 |
| 表示スケール | 0.11 |

#### 十字架配置

```
    ◯      ← 先頭1個（進行方向）
  ◯◯◯    ← 横3個（左・中央・右）
    ◯      ← 後方1個
    ◯      ← 後方2個目
```

※ 「先頭」が常に進行方向を向く（十字架が進行方向に沿って進む）

#### 挙動

1. **詠唱フェーズ**: 0.4秒の詠唱（プレイヤー方向を確定）
2. **突進開始**:
   - 十字架配置の弾幕6発を生成
   - ルーミアが14m/sでプレイヤー方向へ突進
   - 弾幕はルーミアに追従
3. **突進終了**:
   - 7m到達 or プレイエリア端に到達で停止
   - 弾幕は同じ方向・速度（14m/s）で画面外まで継続

#### 備考

- 突進中はルーミア本体にもダメージ判定あり
- 弾幕は突進終了後も14m/sで直進し続ける
- プレイエリア端に到達すると突進距離に関係なく停止
- CC（スタン）でBreak可能、Break時は弾幕も消滅

## Phase1共通仕様

### スキル硬直時間

Phase1（スペルカード中）の各スキル（Q/W/E）使用後、**0.5秒間**は次のスキルを使用できない硬直時間が発生する。
- 移動は可能
- Rスキル（無敵移動）は硬直の影響を受けない
